# sipebiSimpel

Fail basis data ` baku.json ` dibuat berdasarkan data yang ada pada **[Data Terbuka Sipebi](https://docs.google.com/spreadsheets/d/1b6628kbRLqsSM06XuV6iiOnFQXffmfOqMNx_joe_0vQ)**
